# spring-boot-redis-docker-example

### Project Requirements

* JDK 17(openjdk-17)
* Spring Boot 3.0.0
* Maven 4.0.0 or newer
* Docker


### 1.Build Project

    $ git clone https://github.com/matiryaki/spring-boot-redis-docker-example.git
    $ cd redis-example
    $ mvn clean package

#### Run With Docker

    $ docker compose up
